# SQL Analytics Portfolio

This repository contains SQL case study solutions using **BigQuery**, the same tool taught during my data analytics bootcamp.

Each case study uses real data from my bootcamp projects, so I can write questions I genuinely understand and connect the SQL findings to analysis I've already done in Python and Tableau.

## Why these datasets?

Most SQL portfolios use pre-built toy datasets. These two use **real data I already analyzed** during my bootcamp, the same Glassdoor scrape that powers the SalaryIQ Tableau dashboard, and the same OWID CO₂ data from the Indonesia emissions presentation. Using familiar data means I can write meaningful questions, understand the results, and explain the insights clearly.
